# Matrice de l'Ironie Dramatique

## Qui sait quoi ?
La gestion de l'information entre l'Auteur, le Personnage et le Lecteur est ce qui crée l'engagement émotionnel.

Il existe trois états fondamentaux basés sur cette asymétrie d'information.

### 1. Le Mystère (Curiosité)
*   **Équation** : Personnage > Lecteur
*   **Situation** : Le personnage sait quelque chose que le lecteur ignore.
*   **Effet** : Le lecteur est intrigué, il veut savoir. Il tourne les pages pour *comprendre*.
*   *Exemple* : Sherlock Holmes examine un indice et sourit, mais ne dit pas pourquoi.
*   *Danger* : Si cela dure trop longtemps, le lecteur se sent exclu ou frustré.

### 2. Le Suspense (Anxiété)
*   **Équation** : Lecteur > Personnage
*   **Situation** : Le lecteur sait quelque chose que le personnage ignore.
*   **Effet** : Le lecteur a peur pour le personnage. Il tourne les pages pour *voir comment il va s'en sortir*.
*   *Exemple* : On voit le tueur se cacher dans le placard. Le héros entre dans la chambre sans savoir.
*   *Note d'Hitchcock* : C'est la forme la plus puissante d'engagement. "La bombe sous la table".

### 3. La Surprise (Choc)
*   **Équation** : Lecteur = Personnage
*   **Situation** : Personne ne savait. L'information est révélée aux deux en même temps.
*   **Effet** : Choc immédiat, montée d'adrénaline.
*   *Exemple* : "Je suis ton père."
*   *Limitation* : L'effet est bref. Une fois la surprise passée, il faut reconstruire du suspense ou du mystère.

## Tableau de Bord
| État | Qui en sait plus ? | Émotion Cible | Usage |
| :--- | :--- | :--- | :--- |
| **Mystère** | Personnage | Curiosité | Début de chapitre, Enquêtes |
| **Suspense** | Lecteur | Peur / Tension | Scènes d'action, Climax |
| **Surprise** | Égalité | Choc | Twists, Révélations finales |

**Règle d'Or** : Ne restez jamais trop longtemps en état d'égalité (sauf dans les scènes de repos). Basculez constamment entre Mystère et Suspense.
