# Types d'Arcs de Personnages

## 1. L'Arc Positif (The Change Arc)
C'est l'arc le plus classique (ex: Luke Skywalker, Harry Potter).
*   **Début** : Le héros croit en un **Mensonge (Lie)** qui le protège d'une **Blessure (Wound)** passée.
*   **Milieu** : Le monde met ce mensonge à l'épreuve. Le héros souffre car ses vieilles méthodes ne marchent plus.
*   **Fin** : Le héros rejette le Mensonge et embrasse la **Vérité (Truth)**. Il change intérieurement pour vaincre l'antagoniste extérieurement.
*   **Formule** : Lie -> Struggle -> Truth.

## 2. L'Arc Plat (The Flat Arc)
Le héros ne change pas, c'est lui qui change le monde (ex: Superman, Sherlock Holmes, Jack Reacher).
*   **Début** : Le héros connaît déjà la Vérité. Il est compétent et moralement stable.
*   **Milieu** : Le monde (ou les personnages secondaires) résiste à cette Vérité. Le héros est testé dans sa foi, mais ne vacille pas.
*   **Fin** : Le héros a transformé son environnement ou inspiré les autres à changer.
*   **Formule** : Truth -> Test -> Truth (World Changed).

## 3. L'Arc Négatif (The Tragedy)
Le héros échoue à changer ou choisit le mauvais chemin (ex: Anakin Skywalker, Walter White, Macbeth).
*   **Variante A (Désillusion)** : Le héros croit en une Vérité noble, mais le monde est trop corrompu. Il finit par croire au Mensonge (Cynisme).
*   **Variante B (Chute)** : Le héros a un défaut fatal (Lie). Il a l'occasion de changer, mais refuse. Il s'enfonce dans le Mensonge jusqu'à sa destruction.
*   **Formule** : Lie -> Refusal -> Worse Lie (Destruction).

## Matrice de Décision
| Type d'Histoire | Arc Recommandé |
| :--- | :--- |
| Coming of Age, Romance, Hero's Journey | **Positif** |
| Action, Mystère, Super-héros iconique | **Plat** |
| Drame Noir, Horreur psychologique, Biopic tragique | **Négatif** |
